1. Fork this repository.
2. Add your tool to the relevant category table in alphabetical or relevance order.
3. Include: **Tool Name**, **Website**, and a **short, accurate one-line description**.
4. Open a pull request.

Please only submit tools you have verified are active and accurately described. Spam, affiliate-only links, or duplicate entries will be removed.
